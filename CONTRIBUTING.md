# Contributing

## Setup

1. Install Node.js 20+
2. Clone repo
3. Run `npm install`
4. Copy `.env.example` to `.env` and fill values

## Development Workflow

- Run in dev mode: `npm run dev`
- Build: `npm run build`
- Keep changes focused and minimal

## Pull Requests

- Describe behavioral changes clearly
- Include setup or config updates when needed
- Do not include secrets, tokens, or private server identifiers

## Code Style

- TypeScript strict mode is enabled
- Keep logic explicit and defensive around network failures
- Prefer simple, maintainable changes over broad refactors
